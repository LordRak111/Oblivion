#  OBLIVION - Backend API

Серверная часть веб-сайта частной компании в сфере ракетостроения. Реализует REST API для управления новостями, проектами, вакансиями, обработки форм обратной связи и поиска по сайту.

##  Стек технологий
- **Runtime:** Node.js (v20+)
- **Framework:** Express.js
- **Middleware:** `cors`, `dotenv`, `express.json()`
- **Хранение данных:** JSON-файлы (локальное файловое хранилище)
- **Формат ответов:** JSON

##  Установка и запуск
1. Убедитесь, что установлен на компьютере[Node.js](https://nodejs.org/)
2. Перейдите в директорию проекта:
   ```bash
   cd backend
3. Установите зависимости:
   npm install
4. Создать файл конфигурации (на основе шаблона в репозитории):
   cp .env.example .env
5. Запуск сервера:
   npm start
